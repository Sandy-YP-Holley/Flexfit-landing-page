# FlexFit - Product Landing Page

A modern, fully responsive product landing page built using HTML5 and CSS3. This project satisfies all 15 user stories required for the freeCodeCamp Product Landing Page certification lab.

## Features

* **Fixed Navigation:** The header remains at the top of the viewport at all times for easy navigation.
* **Smooth Scrolling:** Clicking navigation links smoothly scrolls to the target sections (Features, How It Works, Pricing).
* **HTML5 Validation:** The email subscription form includes built-in client-side validation.
* **Responsive Layout:** Utilizes CSS Flexbox and media queries to ensure compatibility across mobile, tablet, and desktop screens.
* **Embedded Media:** Includes an optimized, responsive video presentation section.
